## क्या और क्यों?
Bash में वर्तमान दिनांक प्राप्त करना विभिन्न प्रारूपों में दिनांक और समय प्रदर्शित करने के लिए निर्मित कमांड्स का उपयोग शामिल करता है। प्रोग्रामर लॉग्स को समयांकित करने, कार्यों को अनुसूचित करने, या सिस्टम जानकारी स्क्रिप्ट्स के एक हिस्से के रूप में कार्यों को कब किया गया था, ट्रैक करने के लिए इस कार्यक्षमता का उपयोग करते हैं।

## कैसे:
Bash में, `date` कमांड वर्तमान दिनांक और समय प्राप्त करने के लिए आपका प्राथमिक उपकरण है। इसे कैसे उपयोग करें इसके कुछ उदाहरण यहाँ दिए गए हैं:

1. **मुलभूत प्रारूप में वर्तमान दिनांक और समय प्राप्त करें:**

```bash
date
```

*नमूना आउटपुट:*
```
Wed Apr 5 14:22:04 PDT 2023
```

2. **आउटपुट प्रारूप अनुकूलित करें:** आप `+%` प्रारूप निर्दिष्टकों का उपयोग करके आउटपुट प्रारूप को निर्दिष्ट कर सकते हैं। उदाहरण के लिए, YYYY-MM-DD प्रारूप में दिनांक प्रदर्शित करने के लिए:

```bash
date "+%Y-%m-%d"
```

*नमूना आउटपुट:*
```
2023-04-05
```

3. **वर्तमान UNIX टाइमस्टैंप प्राप्त करें:** UNIX टाइमस्टैंप Unix Epoch (जनवरी 1, 1970) से सेकंडों की संख्या है। यह समय अंतरों पर आधारित गणना करने वाली स्क्रिप्ट्स के लिए उपयोगी है।

```bash
date "+%s"
```

*नमूना आउटपुट:*
```
1672877344
```

इस मूलभूत क्रिया के लिए Bash में आमतौर पर कोई लोकप्रिय थर्ड-पार्टी लाइब्रेरीज़ का उपयोग नहीं किया जाता है क्योंकि निर्मित `date` कमांड व्यापक कार्यक्षमता प्रदान करता है। हालांकि, अधिक उन्नत दिनांक और समय संशोधनों के लिए, प्रोग्रामर अन्य प्रोग्रामिंग भाषाओं या उपकरणों का उपयोग कर सकते हैं जो दिनांक अरिथमेटिक और पार्सिंग के लिए लाइब्रेरीज़ प्रदान करते हैं, जैसे कि Python का `datetime` मॉड्यूल।
