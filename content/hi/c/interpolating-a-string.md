## What & Why? (क्या और क्यों?)
स्ट्रिंग इंटरपोलेशन एक ऐसा तरीका है जिसमें हम पूर्व-निर्धारित टेक्स्ट में वेरिएबल्स की वैल्यू जोड़ते हैं। यह डायनामिक मेसेजेस बनाने और कोड को अधिक पढ़ने योग्य बनाने के लिए किया जाता है।

## How to: (कैसे करें:)
C में स्ट्रिंग इंटरपोलेशन सीधा नहीं है, पर `printf` फंक्शन का उपयोग करके हम यह काम कर सकते हैं:

```c
#include <stdio.h>

int main() {
    char name[] = "रोहन";
    int age = 25;

    printf("मेरा नाम %s है और मैं %d वर्ष का हूँ।\n", name, age);
    return 0;
}
```

सैंपल आउटपुट:

```
मेरा नाम रोहन है और मैं 25 वर्ष का हूँ।
```

## Deep Dive (गहराई से जानकारी)
C भाषा में जो आज हम इस्तेमाल करते हैं वह 1972 में विकसित की गयी थी, और स्ट्रिंग इंटरपोलेशन के कई तरीके नहीं हैं। `printf` एक शक्तिशाली फंक्शन है जो विभिन्न प्रकार के प्लेसहोल्डर्स (जैसे `%s` और `%d`) का समर्थन करता है।

अन्य भाषाएं, जैसे कि Python या JavaScript, मॉडर्न स्ट्रिंग इंटरपोलेशन फीचर्स प्रदान करती हैं जैसे कि f-strings और template literals। हालांकि, C में इसे अधिक सीधा तरीके से करने के लिए `sprintf` या `snprintf` जैसे फंक्शन्स का इस्तेमाल किया जा सकता है, जो स्ट्रिंग्स को बफ़र में स्टोर कर देते हैं।
