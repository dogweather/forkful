## Что и Почему?

Запись текстового файла подразумевает сохранение данных в файле, содержащем текст, обычно в формате, доступном для чтения человеком, таком как `.txt` или `.csv`. Программисты записывают файлы для сохранения и сохранения данных на будущее, которые могут читаться человеком или использоваться другими программами позже.

## Как это сделать:

Вот как вы пишете строку в текстовый файл на Go:

```Go
package main

import (
	"log"
	"os"
)

func main() {
	message := "Привет, Go!"

	file, err := os.Create("example.txt")
	if err != nil {
		log.Fatal(err)
	}
	defer file.Close()

	_, err = file.WriteString(message)
	if err != nil {
		log.Fatal(err)
	}

	log.Println("Файл успешно записан!")
}
```

Запустите его. Если все прошло успешно, ошибок не будет, и создастся `example.txt`.

## Погружение

Запись в текстовые файлы на Go использует пакет `os`, который предоставляет платформонезависимый интерфейс к функциональности операционной системы. Функция `os.Create` создает или обрезает файл. Метод `File.WriteString` прост и нагляден для записи строк.

Исторически обработка текстовых файлов эволюционировала из библиотеки `stdio.h` языка C. В Go простота является ключевым моментом; вы делаете меньше для большего действия, избегая шаблонного кода. Существуют альтернативы, например `ioutil.WriteFile`, но они не рекомендуются для больших файлов из-за неэффективности использования памяти. `bufio` предоставляет буферизованный ввод-вывод, уменьшая количество системных вызовов и повышая производительность.
