## Что и Почему?

Конкатенация строк — это их соединение конец в конец для формирования новой строки. Программисты делают это для динамического объединения текста, например, при составлении сообщений или генерации путей к файлам.

## Как это делается:

В C для конкатенации строк используется функция `strcat` из `string.h`. Но будьте внимательны, вам нужен достаточно большой массив назначения, чтобы он мог вместить результат объединения.

```C
#include <stdio.h>
#include <string.h>

int main() {
    char destination[50] = "Привет, ";
    char source[] = "мир!";

    // Конкатенация `source` в `destination`
    strcat(destination, source);

    // Вывод сконкатенированной строки
    printf("%s\n", destination); // "Привет, мир!"

    return 0;
}
```

Убедитесь, что ваш массив назначения не переполнится. Это ваша задача, так как C не сделает это за вас.

## Погружение

Конкатенация является базовой операцией с текстом с самого начала вычислений. В C функции вроде `strcat` и `strncat` (которая ограничивает количество конкатенируемых символов) выполняют основную работу. C не управляет памятью за вас, поэтому помните о необходимости выделения достаточного объема памяти перед конкатенацией.

Альтернативы? О, конечно. Если вас беспокоят переполнения буфера, вы можете использовать вместо этого `snprintf`. Это безопаснее, поскольку позволяет указать максимальный размер выходного буфера:

```C
char buffer[50];
snprintf(buffer, 50, "%s%s", "Привет, ", "мир!");
```

Что касается деталей, `strcat` работает, находя конец первой строки и копируя туда вторую строку символ за символом. Просто, но ручное управление памятью делает его подверженным ошибкам, таким как переполнения буфера.
