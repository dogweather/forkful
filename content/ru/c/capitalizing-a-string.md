## Что и Почему?
Преобразование строки в заглавные буквы означает превращение всех строчных букв в прописные. Программисты часто используют преобразование строк в заглавные буквы для обеспечения единообразия, форматирования отображения или как часть процессов нормализации данных.

## Как это сделать:
В языке C нет встроенной функции для преобразования строк в заглавные буквы. Обычно вы проходите циклом по каждому символу, преобразуя его по мере продвижения:

```c
#include <stdio.h>
#include <ctype.h>

void capitalizeString(char *str) {
    while (*str) {
        *str = toupper((unsigned char) *str);
        str++;
    }
}

int main() {
    char myString[] = "hello world!";
    capitalizeString(myString);
    printf("%s\n", myString);  // Вывод: HELLO WORLD!
    return 0;
}
```

## Погружение в детали
В ранние дни компьютерной эры операции со строками были простыми и ручными. C, разработанный в начале 1970-х годов, отражает это с помощью простых функций для манипуляции со строками в его стандартной библиотеке. Функция `toupper` предназначена для преобразования одного символа в прописную букву. Она является частью `<ctype.h>`, заголовочного файла, содержащего функции для проверки и преобразования символов.

Существуют альтернативы прохождению циклом по строке для ее преобразования в заглавные буквы. Библиотеки, такие как `libCStringUtils`, предлагают более сложные операции со строками, включая преобразование в заглавные буквы. Некоторые разработчики также пишут свои собственные функции с такими особенностями, как учет локализации.

Внутренне, символы ASCII имеют числовые эквиваленты, которые отличаются на 32 между прописными и строчными буквами. Функция `toupper` использует это различие для преобразования символов. Однако прямая работа с ASCII значениями нежелательна из-за проблем с читаемостью и локализацией.
