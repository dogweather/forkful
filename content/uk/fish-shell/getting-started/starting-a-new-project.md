## What & Why? (Що і Чому?)
Започаткування нового проєкту — це створення основи для вашого коду. Програмісти роблять це, аби організувати думки та структуру до написання першого рядка коду.

## How to: (Як зробити:)
```Fish Shell
# Створити папку для нового проєкту
mkdir my_new_project

# Перейти у папку
cd my_new_project

# Ініціалізація Git репозиторіуму
git init

# Створити новий файл 'main.fish'
touch main.fish

# Відкрити файл у текстовому редакторі
code main.fish
```
```Output
Initialized empty Git repository in /your/path/my_new_project/.git/
```

## Deep Dive (Поглиблений Аналіз)
Започаткування проєкту було важливим кроком з самого початку програмування. У минулому, сетап був складнішим, з більшою кількістю кроків та налаштувань. Але із часом, інструменти як Git стали стандартом, спрощуючи процес. Риболовля (Fish) — молодший серед оболонок, порівняно з bash чи zsh, але набула популярності завдяки своїй простоті та потужності. Вона відмінно підходить для скриптингу та автоматизації завдань, таких як започаткування нового проєкту.

Альтернативами можуть бути генератори схем, як Yeoman, або інструменти для створення контейнерів, такі як Docker, якщо вам потрібно більше ізоляції та контролю.

Щодо реалізації в Fish, то варто зазначити, що ця оболонка пропонує багато вбудованих функцій та зручних команд, що звільняють вас від складних шаблонів і дають змогу фокусуватися на основному завданні.
