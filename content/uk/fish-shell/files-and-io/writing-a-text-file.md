## Що і чому?

Запис у текстовий файл у Fish Shell дозволяє зберігати дані у постійному вигляді, що забезпечує легке отримання або маніпулювання даними як за допомогою того ж Fish скрипту, так і інших програм. Програмісти роблять це для логування, зберігання налаштувань конфігурації або експорту даних для подальшої обробки.

## Як:

Щоб записати у текстовий файл у Fish, ви можете використовувати команду `echo` разом з операторами перенаправлення. Немає популярних сторонніх бібліотек спеціально для запису у файл у Fish, оскільки вбудовані команди оболонки є простими та ефективними для цієї мети.

### Запис тексту до нового файлу або перезапис існуючого файлу:
```fish
echo "Привіт, Fish Shell!" > output.txt
```
Ця команда записує "Привіт, Fish Shell!" у `output.txt`, створюючи файл, якщо він не існує, або перезаписуючи його, якщо він існує.

### Додавання тексту до існуючого файлу:
Якщо ви хочете додати текст у кінець існуючого файлу без видалення його поточного вмісту, використовуйте оператор додавання `>>`:
```fish
echo "Додаю новий рядок до файлу." >> output.txt
```

### Запис кількох рядків:
Ви можете записати кілька рядків у файл, використовуючи echo з символом нового рядка `\n`, або ви можете з\'єднати кілька команд echo за допомогою крапок з комою:
```fish
echo "Перший Рядок\nДругий Рядок" > output.txt
# АБО
echo "Перший Рядок" > output.txt; echo "Другий Рядок" >> output.txt
```

### Приклад виводу:
Щоб переглянути вміст `output.txt` після виконання вищеописаних команд, використовуйте команду `cat`:
```fish
cat output.txt
```
```plaintext
Перший Рядок
Другий Рядок
```
Заміна або додавання текстів, як показано, маніпулює вмістом файлу відповідно до ваших вимог, демонструючи прості, але потужні способи роботи з текстовими файлами у Fish Shell.
