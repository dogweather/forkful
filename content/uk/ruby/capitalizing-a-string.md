## Що це та навіщо?

Капіталізація рядка - це процес перетворення першої літери рядка у велику букву. Програмісти використовують це для форматування тексту виводу або для відповідності різним конвенціям стилю.

## Як це зробити:

Ruby досить просто капіталізувати рядок. Ось приклад:

```Ruby
string = "привіт"
puts string.capitalize
```

Output:

```
Привіт
```

Ось більш складний приклад коду:

```Ruby
string = "привіт, світе"
puts string.split.map(&:capitalize).join(' ')
```

Output:

```
Привіт, Світе
```

## Більш глибокий погляд

Капіталізація рядків існує вже давно і була частиною багатьох мов програмування, ще до Ruby. Різні мови можуть мати власні функції для капіталізації, і вони можуть працювати трохи по-різному, залежно від деталей реалізації.

В Ruby, метод `capitalize` працює тільки з ASCII символами, тому для роботи з Unicode символами може знадобитись додатковий код або використання бібліотек, таких як ActiveSupport.

Також можна використовувати `upcase` для перетворення всього рядка у верхній регістр або `downcase` для перетворення у нижній регістр.
