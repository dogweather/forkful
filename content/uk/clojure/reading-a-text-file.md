## Що і чому?

Читання текстових файлів - це процес отримання даних із документу для подальшого використання в програмі. Програмісти цим займаються, щоб обробляти структуровані дані (як-от логи серверів або дані CSV) та аналізувати текст.

## Як це зробити:

Використовуємо функцію `slurp` для читання файла. Приклад:

```Clojure
(def text (slurp "myfile.txt"))
(println text)
```

Вираз `(slurp "myfile.txt")` повертає вміст myfile.txt як рядок, і цей рядок тоді виводиться на консоль.

## Поглиблений погляд

Clojure - сучасна версія мови Lisp, яка вперше з'явилася в 1958 році, а її синтаксис відомий багаторядковими виразами для зчитування даних з файлів.

У Clojure ви також можете використовувати `line-seq` для читання файлу построково. Це може бути корисним, якщо файл надто великий, щоб вмістити його в пам'ять цілком.

Використання `slurp`, з іншого боку, простіше та швидше, коли працюєте з меншими файлами. Файл повністю зчитується в пам'ять в міг, і вам не потрібно займатися буферизацією або особливим кодом для обробки помилок ввід/вивід.
