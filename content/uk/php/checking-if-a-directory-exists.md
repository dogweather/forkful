## What & Why? / Що і чому?

Перевірка існування директорії у PHP - це спосіб забезпечити, що ваш скрипт не впаде, спробувавши доступитися до неіснуючого каталогу. Програмісти роблять це, щоб уникнути помилок під час роботи з файлами та директоріями.

## How to: / Як робити:

```PHP
<?php
// Перевірка існування директорії
$directory = "/path/to/the/directory";

if (is_dir($directory)) {
    echo "Директорія існує!";
} else {
    echo "Директорії не існує.";
}

// Приклад виводу, якщо директорія існує
// Директорія існує!
?>
```

## Deep Dive / Поглиблено:

Перевірка існування директорій в PHP існує з моментів ранніх версій мови. Функція `is_dir()` перевіряє, чи існує директорія і чи це взагалі директорія. Альтернативою є `file_exists()`, яка перевіряє існування файлів та каталогів, але якщо вам потрібно бути впевненим, що це саме директорія, `is_dir()` кращий вибір. Реалізація перевірки може бути простою, як у прикладі вище, але може включати і створення директорії, якщо вона не існує, з допомогою `mkdir()`.
