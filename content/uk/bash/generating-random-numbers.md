**## Що й навіщо?**

Генерація випадкових чисел - це формування числових значень, які з'являються незалежно від попередніх результатів. Програмісти роблять це для введення елементу непередбачуваності в програми, наприклад, для тестування, ігор, шифрування й т. д.

**## Як це зробити:**

Тут наведено код та приклади виведення, за допомогою яких ви зможете генерувати випадкові числа в Bash.

```Bash
# Генерація випадкового числа від 0 до 9
echo $RANDOM % 10
```

```Bash
# Приклад виведення
8
```

Зауважте, що `RANDOM` - це вбудована змінна у Bash, яка автоматично повертає нове випадкове число кожного разу, коли ви її викликаєте.

**## Докладніше:**

Перші комп'ютерні програми для генерації випадкових чисел були створені в середині 20 століття. Генерація випадкових чисел в Bash відбувається за допомогою pseudorandom number generator (PRNG), що впливає на максимально допустиме значення для `RANDOM` (32767).

Альтернативою може бути використання спеціалізованих засобів, таких як `openssl`, чи зчитування значень з `/dev/random` або `/dev/urandom`.

```Bash
# Отримати випадкове число від 0 до 9 використовуючи openssl
openssl rand -rand /dev/urandom 1 | od -An -i | awk -v f=0 -v r=9 '{printf "%i\n", ($1 % (r-f+1)+f)}'
```
