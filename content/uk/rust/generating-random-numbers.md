## Що і чому?
Генерація випадкових чисел - це процес створення чисел, які виходять за рамки явного прогнозування, зазвичай через алгоритми. Це використовується у програмуванні для реалізації функціональності, яка потребує випадковості або подачі, наприклад, в іграх, шифруванні або в дослідницькій статистиці. 

## Як це зробити:
Використовуймо край за допомогою бібліотеки `rand`. Давайте згенеруємо випадкове число в інтервалі від 1 до 10.

```Rust
fn main() {
    let mut rng = rand::thread_rng();
    let n = rng.gen_range(1..=10);
    println!("Випадкове число: {}", n);
}
```
Виходить:
```
Випадкове число: 7
```
## Поглиблений огляд:
У багатьох випадках, когда програмисти можуть вважати за необхідне використовувати генерацію випадкових чисел, було б важко уявити світ інформаційних технологій без неї. Генерація псевдовипадкових чисел, была вперше задокументована в далекому 1946 році, і відтоді розширюється та вдосконалюється.

Сьогодні існує декілька альтернативних методів для генерації випадкових чисел в Rust, включаючи `OsRng`, `EntropyRng` та `StdRng`.

Деталі щодо реалізації генерації випадкових чисел залежать від ваших власних потреб та контексту. Однак важливо згадати, що генератор випадкових чисел лише створює ілюзію випадковості, і не слід використовувати ці числа для критичних застосувань, маючі на увазі безпеку.
