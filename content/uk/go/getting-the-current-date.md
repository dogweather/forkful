## Що і чому?

Отримання поточної дати – це процес зчитування системного часу у вашій програмі. Програмісти роблять це, щоб відмічати події, логувати зміни, або для функцій, що залежать від дати.

## Як робити:

Отримання поточної дати в Go виглядає так:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    currentDate := time.Now()
    fmt.Println("Current Date and Time is:", currentDate)
}
```

При запуску цього коду отримаєте вивід схожий на:

```
Current Date and Time is: 2023-04-01 15:04:05.123456 +0300 EEST
```

## Поглиблено:

Вже давно програми потребують відстежування часу. Традиційно, це було асоційоване із системними годинниками. В Go, це виконується через пакет `time`, який надає інструменти для роботи з датою та часом.

Альтернативи? Бібліотеки сторонніх розробників, але пакет `time` є достатньо потужним для більшості завдань і вбудований прямо у мову.

Деталі реалізації? `time.Now()` внутрішньо використовує системні виклики, щоб отримати поточний час, і потім видає його у форматі об'єкта `Time`. Цей об'єкт можна форматувати, розбирати, порівнювати та виконувати з ним інші дії.
