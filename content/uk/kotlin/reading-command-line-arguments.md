## Що і чому?

Читання аргументів командного рядка - це процес отримання даних, що передаються програмі під час її запуску через командний рядок. Програмісти роблять це, щоб забезпечити гнучкість та масштабованість додатків.

## Як це робити:

```Kotlin
fun main(args: Array<String>) {
    for (arg in args) {
        println(arg)
    }
}
```
Якщо ви запустите цю програму з аргументами командного рядка, наприклад `kotlin MainKt Hello Ukraine`, вона виведе:
  
```
Hello
Ukraine
```

## Поглиблений огляд

Історично, читання аргументіv командного рядка було важливим для роботи з системами в реальному часі, де користувачам потрібно було робити величезну кількість задач за допомогою командного рядка. Є альтернативи, такі як введення даних через GUI або файл konfigурації, але в більшості випадків аргументи командного рядка повертаються до масштабованості та ефективності.

Компілятор Kotlin автоматично забезпечує `args: Array<String>` до вашої `main`-функції, якщо ви не визначаєте її самостійно. Кожен елемент `args` масиву є стрічкою, незалежно від типу введення командного рядка.
