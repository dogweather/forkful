## কিভাবে:
1. **Dart ইনস্টল করুন**:
   নিশ্চিত করুন যে আপনার সিস্টেমে Dart ইনস্টল করা আছে। যদি না থাকে, আপনি [https://dart.dev/get-dart](https://dart.dev/get-dart) থেকে এটি ডাউনলোড করতে পারেন। ইনস্টলেশন যাচাই করুন:

   ```shell
   dart --version
   ```

2. **একটি নতুন Dart প্রকল্প তৈরি করুন**:
   একটি নতুন প্রকল্প উৎপাদনের জন্য Dart CLI ব্যবহার করুন:

   ```shell
   dart create hello_dart
   ```

   এই কমান্ডটি একটি নতুন ডিরেক্টরি `hello_dart` তৈরি করে যা আপনার নির্বাচন অনুযায়ী একটি সাধারণ নমুনা ওয়েব বা কনসোল অ্যাপ্লিকেশন ধারণ করে।

3. **প্রকল্প কাঠামো পরীক্ষা করুন**:
   
   আপনার প্রকল্প ডিরেক্টরিতে নেভিগেট করুন:

   ```shell
   cd hello_dart
   ```

   একটি টিপিকাল Dart প্রকল্প নিম্নলিখিত মুখ্য ফাইল এবং ডিরেক্টরিগুলি ধারণ করে:

   - `pubspec.yaml`: আপনার প্রকল্পের ডিপেন্ডেন্সিগুলি এবং SDK বাঁধাধরা অন্তর্ভুক্ত করে এমন কনফিগারেশন ফাইল।
   - `lib/`: ডিরেক্টরি যেখানে বেশিরভাগ Dart কোড অবস্থান করে।
   - `test/`: প্রকল্পের টেস্টগুলির জন্য ডিরেক্টরি।

4. **ডিপেন্ডেন্সিগুলি যোগ করুন**:
   ডিপেন্ডেন্সিগুলি যোগ করার জন্য `pubspec.yaml` সম্পাদনা করুন। ওয়েব প্রকল্পের জন্য, HTTP অনুরোধ করা একটি জনপ্রিয় প্যাকেজ `http` যোগ করা বিবেচনা করুন:

   ```yaml
   dependencies:
     flutter:
       sdk: flutter
     এইচটিটিপি: ^0.13.3
   ```

   সম্পাদনা করার পর, ডিপেন্ডেন্সিগুলি পান:

   ```shell
   dart pub get
   ```

5. **আপনার প্রথম Dart কোড লিখুন**:
   
   `lib/` ডিরেক্টরিতে, একটি নতুন Dart ফাইল, `main.dart` তৈরি করুন, এবং একটি সাধারণ Dart কোড যোগ করুন:

   ```dart
   // Dart কোর লাইব্রেরি ইম্পোর্ট করুন
   import 'dart:core';

   void main() {
     print('Hello, Dart!');
   }
   ```

6. **আপনার Dart অ্যাপ্লিকেশন রান করুন**:

   আপনার Dart প্রোগ্রাম নিষ্পাদন করুন সঙ্গে:

   ```shell
   dart run
   ```

   আউটপুট হবে:

   ```
   Hello, Dart!
   ```

এই ধাপগুলি অনুসরণ করে, আপনি সফলভাবে একটি নতুন Dart প্রকল্প শুরু করেছেন, ইনস্টলেশন থেকে আপনার প্রথম Dart কোড চালানো পর্যন্ত। এই মৌলিক জ্ঞান আপনাকে Dart-এর সমৃদ্ধ ইকোসিস্টেমে গভীরে ডুব দিতে এবং স্কেলেবল অ্যাপ্লিকেশন তৈরিতে এর সক্ষমতার প্রয়োগের জন্য প্রস্তুত করে।
