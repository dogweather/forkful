## কি এবং কেন?

C ভাষায় নির্দিষ্ট প্যাটার্নে মিলে যাওয়া অক্ষরগুলি স্ট্রিং থেকে মুছে ফেলার প্রক্রিয়া হলো নির্দিষ্ট কিছু অক্ষর যা প্রাকডিফাইনড মানদণ্ডে মিলে যায়, সেগুলির সবকটি উদাহরণ মুছে ফেলা। প্রোগ্রামাররা ইনপুট সেনিটাইজ করা, ডাটা প্রক্রিয়াজন্য প্রস্তুত করা, অথবা কেবল স্ট্রিংগুলি আউটপুটের জন্য বা আরও ম্যানিপুলেশনের জন্য পরিষ্কার করা ইত্যাদি কাজে এই কাজ সম্পাদন করেন, এতে করে ডাটা যাতে ঠিক সেই প্রয়োজনীয় কন্টেক্সট অথবা অ্যালগোরিদমের জন্য ঠিক মতো থাকে তা নিশ্চিত করা হয়।

## কিভাবে:

C সরাসরি প্যাটার্নের উপর ভিত্তি করে স্ট্রিং থেকে চরিত্রগুলি মুছে ফেলার জন্য কোনো অন্তর্নির্মিত ফাংশন সহ আসে না, যা কিছু উচ্চ মাত্রার ভাষায় রয়েছে। তবে, আপনি স্ট্রিংটির উপর ম্যানুয়ালি ইটারেটিং করে এবং অনাকাঙ্ক্ষিত অক্ষরগুলি বাদ দিয়ে একটি নতুন স্ট্রিং তৈরি করে সহজেই এই কাজটি সম্পাদন করতে পারেন। উদাহরণ স্বরূপ, ধরা যাক আপনি একটি স্ট্রিং থেকে সকল ডিজিট মুছে ফেলতে চান। আপনি এটি নিম্নলিখিতভাবে করতে পারেন:

```c
#include <stdio.h>
#include <ctype.h>

void remove_digits(char *str) {
    char *src = str, *dst = str;
    while (*src) {
        if (!isdigit((unsigned char)*src)) {
            *dst++ = *src;
        }
        src++;
    }
    *dst = '\0';
}

int main() {
    char str[] = "C Programming 101: The Basics!";
    remove_digits(str);
    printf("Result: %s\n", str);
    return 0;
}
```

নমুনা আউটপুট:
```
Result: C Programming : The Basics!
```

এই উদাহরণে `isdigit` ফাংশনটি `ctype.h` থেকে ডিজিটগুলি চিহ্নিত করে, স্ট্রিংয়ের শুরুতে নন-ডিজিট চরিত্রগুলির অবস্থান পরিবর্তন করে, এবং সকল চরিত্র মূল্যায়ন করা হলে স্ট্রিং শেষ করে।
