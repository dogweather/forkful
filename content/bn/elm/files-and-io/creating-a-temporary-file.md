# কি এবং কেন?

অস্থায়ী ফাইল তৈরি করা মানে এমন একটি ফাইল তৈরি করা যা স্বল্প মেয়াদের ব্যবহারের জন্য নির্মিত। প্রোগ্রামাররা এটি করে থাকেন যেমন সংবেদনশীল তথ্য সুরক্ষার জন্য অথবা কোন প্রক্রিয়াকালে অন্তবর্তী ফলাফল পরিচালনা করার জন্য।

# কিভাবে:

এলম ব্রাউজারে চালায়, তাই এর সরাসরি ফাইলসিস্টেমের প্রবেশাধিকার নেই। সুতরাং, আপনি পারম্পরিক অস্থায়ী ফাইল তৈরি করতে পারবেন না। কিন্তু, যদি আপনার এরকম একটি বৈশিষ্ট্যের প্রয়োজন হয়, তাহলে আমরা জাভাস্ক্রিপ্টের সাথে মিথস্ক্রিয়া করার জন্য এলম পোর্টস ব্যবহার করি, যা অস্থায়ী ফাইল তৈরি করতে পারে।

```elm
port module Main exposing (..)

-- জাভাস্ক্রিপ্টে অস্থায়ী ফাইল তৈরির জন্য একটি পোর্ট নির্ধারণ করা
port createTempFile : String -> Cmd msg

-- জাভাস্ক্রিপ্টে অস্থায়ী ফাইল তৈরি করার জন্য ডাটা পাঠানো
saveDataTemporarily : String -> Cmd msg
saveDataTemporarily data =
    createTempFile data
```

জাভাস্ক্রিপ্টের অংশের জন্য, File API ব্যবহার করে:

```javascript
app.ports.createTempFile.subscribe(function(data) {
    var blob = new Blob([data], {type: 'text/plain'});
    var url = URL.createObjectURL(blob);

    // এখানে আপনি ব্লবটি ডাউনলোড করতে অথবা আপনার অ্যাপের অন্যান্য অংশের সাথে পাস করতে URL ব্যবহার করতে পারেন
    console.log(url);  // এটি অস্থায়ী ফাইলের URL লগ করে
});
```

জাভাস্ক্রিপ্ট কনসোলে নমুনা আউটপুট:

```plaintext
blob:null/2135a9b7-1aad-4e7a-8bce-19c4f3f6d7ff
```

# গভীরে যাওয়া

এলম নিরাপদ এবং বিশ্বস্ত হওয়ার জন্য নকশা করা হয়েছে, তাই সরাসরি ফাইল সিস্টেম প্রবেশাধিকার এটির মধ্যে নেই। পরিবর্তে, এলম জাভাস্ক্রিপ্টের সাথে মিথস্ক্রিয়া করতে পোর্টস ব্যবহার করে, যেমন অস্থায়ী ফাইল তৈরির মতো অপারেশনগুলি সম্ভব হয়। ঐতিহাসিকভাবে, ব্রাউজার মাধ্যমে ফাইল-ভিত্তিক কাজগুলিকে জাভাস্ক্রিপ্ট APIs ব্যবহার করে পরিচালনা করা হয়েছে, টাইপ-নিরাপদ, উচ্চ-স্তরের যৌক্তিকতা জন্য এলম ব্যবহার করে।

ওয়েবঅ্যাসেম্বলির মতো বিকল্পগুলি ভবিষ্যতে ফাইল সিস্টেমের সাথে আরও সরাসরি মিথস্ক্রিয়া করার অনুমতি দিতে পারে, কিন্তু এখন পর্যন্ত, জাভাস্ক্রিপ্টের সাথে ইন্টারপ প্রচলিত অনুশীলন।

বাস্তবায়নের দিক থেকে, ব্রাউজার পরিসরে অস্থায়ী ফাইল তৈরি করা মানে বাস্তব ফাইলসিস্টেমে একটি আসল ফাইল নয়, বরং একটি মেমোরিতে আকারে প্রতিনিধিত্ব (ব্লব) যা আপনি প্রয়োজন অনুযায়ী সংরক্ষণ এবং কাজে লাগাতে পারেন।
