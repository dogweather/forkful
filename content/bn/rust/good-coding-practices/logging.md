## কিভাবে:
চলুন আমরা Rust ব্যবহার করে একটি বেসিক লগিং সিনারিও সেট আপ করি, `log` ক্রেট ব্যবহার করে, যা একটি লগিং ফ্যাকেড প্রদান করে, এবং `env_logger`, `log` ক্রেটের জন্য একটি লগিং বাস্তবায়ন। প্রথমে, তাদেরকে আপনার Cargo.toml এ যোগ করুন:

```toml
[dependencies]
log = "0.4.14"
env_logger = "0.9.0"
```

এখন, আপনার `main.rs` এ লগার সেট আপ এবং ইনিশিয়ালাইজ করুন:

```rust
use log::{info, warn};

fn main() {
    env_logger::init();

    info!("এটি একটি ইনফো মেসেজ।");
    warn!("এটি একটি ওয়ার্নিং মেসেজ।");
}
```

আপনার অ্যাপ চালান `RUST_LOG=info cargo run` দিয়ে, এবং আপনি আউটপুট দেখবেন:

```
INFO: এটি একটি ইনফো মেসেজ।
WARN: এটি একটি ওয়ার্নিং মেসেজ।
```

`RUST_LOG` এনভায়রনমেন্ট ভ্যারিয়েবল নিয়ে খেলুন এটি `error`, `warn`, `info`, `debug`, বা `trace` সেট করে আপনার লগগুলির বাচ্যসূচকতা নিয়ন্ত্রণ করে।
