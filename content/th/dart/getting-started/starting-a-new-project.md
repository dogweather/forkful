## อะไร & ทำไม?

การเริ่มต้นโปรเจกต์ใหม่ใน Dart มีการตั้งค่าสภาพแวดล้อมที่ส่งเสริมให้พัฒนาการทดสอบ และการประจำการได้อย่างมีประสิทธิภาพ นักโปรแกรมมักสร้างโปรเจกต์ Dart ใหม่เพื่อใช้ประโยชน์จากประสิทธิภาพที่เหมาะสมและระบบนิเวศที่แข็งแกร่งของ Dart โดยเฉพาะสำหรับการพัฒนาแอปบนเว็บและมือถือด้วยกรอบการทำงานเช่น Flutter

## วิธีการ:

1. **ติดตั้ง Dart**:
   ตรวจสอบให้แน่ใจว่า Dart ได้ถูกติดตั้งไว้ในระบบของคุณแล้ว หากยังไม่ได้ติดตั้ง คุณสามารถดาวน์โหลดได้จาก [https://dart.dev/get-dart](https://dart.dev/get-dart) ตรวจสอบการติดตั้งด้วยคำสั่ง:

   ```shell
   dart --version
   ```

2. **สร้างโปรเจกต์ Dart ใหม่**:
   ใช้ Dart CLI เพื่อสร้างโปรเจกต์ใหม่:

   ```shell
   dart create hello_dart
   ```

   คำสั่งนี้จะสร้างไดเรกทอรีใหม่ `hello_dart` พร้อมแอปตัวอย่างง่ายๆ สำหรับเว็บหรือแอปพลิเคชันคอนโซล ขึ้นอยู่กับการเลือกของคุณ

3. **ตรวจสอบโครงสร้างโปรเจกต์**:
   
   นำทางไปยังไดเรกทอรีโปรเจกต์ของคุณ:

   ```shell
   cd hello_dart
   ```

   โปรเจกต์ Dart ทั่วไปประกอบด้วยไฟล์และไดเรกทอรีหลักดังต่อไปนี้:

   - `pubspec.yaml`: ไฟล์การกำหนดค่าที่รวมกลุ่มเพิ่มเติมและข้อจำกัดของ SDK ของโปรเจกต์คุณ 
   - `lib/`: ไดเรกทอรีที่มีโค้ด Dart ส่วนใหญ่อยู่
   - `test/`: ไดเรกทอรีสำหรับการทดสอบโปรเจกต์

4. **เพิ่ม Dependencies**:
   แก้ไข `pubspec.yaml` เพื่อเพิ่ม dependencies สำหรับโปรเจกต์เว็บ ให้พิจารณาเพิ่ม `http` ซึ่งเป็นแพคเกจยอดนิยมสำหรับการทำ HTTP requests:

   ```yaml
   dependencies:
     flutter:
       sdk: flutter
     http: ^0.13.3
   ```

   หลังจากแก้ไข เรียก dependencies ด้วยคำสั่ง:

   ```shell
   dart pub get
   ```

5. **เขียนโค้ด Dart ของคุณเป็นครั้งแรก**:
   
   ในไดเรกทอรี `lib/` สร้างไฟล์ Dart ใหม่ `main.dart` และเพิ่มโค้ด Dart ง่าย ๆ:

   ```dart
   // นำเข้า Dart core library
   import 'dart:core';

   void main() {
     print('Hello, Dart!');
   }
   ```

6. **รันแอปพลิเคชัน Dart ของคุณ**:

   รันโปรแกรม Dart ของคุณด้วยคำสั่ง:

   ```shell
   dart run
   ```

   ผลลัพธ์จะเป็น:

   ```
   Hello, Dart!
   ```

โดยการทำตามขั้นตอนเหล่านี้ คุณได้เริ่มต้นโปรเจกต์ Dart ใหม่สำเร็จ ตั้งแต่การติดตั้งจนถึงการรันโค้ด Dart แรกของคุณ ความรู้พื้นฐานนี้จะเป็นพื้นฐานให้คุณสามารถขุดลึกลงไปในระบบนิเวศและความสามารถของ Dart ในการสร้างแอปพลิเคชันที่ขยายได้ง่ายอย่างเต็มศักยภาพ.
