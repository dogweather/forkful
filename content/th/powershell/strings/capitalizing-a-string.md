## อะไรและทำไม?
การทำให้ตัวอักษรแรกของสตริงเป็นตัวพิมพ์ใหญ่ใน PowerShell หมายถึงการเปลี่ยนตัวอักษรตัวแรกของสตริงที่กำหนดให้เป็นตัวพิมพ์ใหญ่ ขณะที่ข้อความที่เหลือในสตริงยังคงเหมือนเดิม โปรแกรมเมอร์มักจะทำงานนี้เพื่อวัตถุประสงค์ในการจัดรูปแบบ เช่น เตรียมข้อความเพื่อแสดงในหน้าต่างของผู้ใช้หรือปฏิบัติตามกฎไวยากรณ์ในเอกสารที่สร้างขึ้น

## วิธีการ:
PowerShell เป็นเครื่องมือที่หลากหลาย ช่วยให้คุณสามารถทำให้สตริงเป็นตัวพิมพ์ใหญ่ได้โดยใช้วิธีการที่ตรงไปตรงมาโดยไม่ต้องใช้ไลบรารีภายนอก นี่คือวิธีที่คุณสามารถทำได้:

```powershell
# ใช้เมทอดในตัวของ .Net 'ToTitleCase' จาก CultureInfo
$text = "hello world"
$culture = [System.Globalization.CultureInfo]::InvariantCulture
$capitalizedText = $culture.TextInfo.ToTitleCase($text.ToLower())
Write-Output $capitalizedText
```
ผลลัพธ์:
```
Hello world
```

หมายเหตุ: วิธีนี้ทำให้ตัวอักษรตัวแรกของแต่ละคำเป็นตัวพิมพ์ใหญ่ ถ้าคุณต้องการทำให้เพียงตัวอักษรตัวแรกของสตริงเป็นตัวพิมพ์ใหญ่และทิ้งส่วนที่เหลือเอาไว้เหมือนเดิม คุณสามารถทำได้โดยสิ่งนี้:

```powershell
# ทำให้เพียงตัวอักษรตัวแรกของสตริงเป็นตัวพิมพ์ใหญ่
$text = "hello world"
$capitalizedText = $text.Substring(0,1).ToUpper() + $text.Substring(1)
Write-Output $capitalizedText
```
ผลลัพธ์:
```
Hello world
```

PowerShell ไม่รวมฟังก์ชันที่ง่ายสำหรับการทำให้เพียงตัวอักษรตัวแรกของสตริงเป็นตัวพิมพ์ใหญ่โดยตรง แต่โดยการรวมวิธีการจัดการสตริงพื้นฐาน เช่น `Substring(0,1).ToUpper()` และการต่อสตริง เราสามารถบรรลุผลลัพธ์ที่ต้องการได้อย่างง่ายดาย
