## מה ולמה?
הפיכת מחרוזת לאותיות רישיות היא פשוט לתת לכל האותיות הקטנות במילה להתחיל באות גדולה. תכניתנים עושים זאת לשיפור קריאות, להדגיש כותרות או שמות פרטיים.

## איך לעשות:
גלים הוא שפה מעולה למשימות טקסט. הנה איך ממירים מחרוזת לרישיות:

```gleam
fn capitalize_string(str: String) -> String {
  str
  |> string.split("") // פוצל את המחרוזת לרשימת תווים
  |> list.map(string.uppercase) // הפוך כל תו לרישי
  |> string.join("") // הרכב בחזרה למחרוזת
}

fn main() {
  let my_string = "שלום לך, עולם!"
  let capitalized = capitalize_string(my_string)
  io.println(capitalized)
}
```

פלט לדוגמא:
```
"שלום לך, עולם!"
```

## צלילה לעומק
היסטורית, הפיכת מחרוזות לגדולות הייתה חלק מהדפסת טקסטים ועיבוד נתונים. באופן מסורתי, שפות תכנות כגון C דרשו לולאות ידניות ובדיקות תנאים. היום, שפות כמו גלים מספקות ספריות ידידותיות שפשטו את התהליך.

עוד דרכים להפוך לרישי: במקום להשתמש בגישה של להפוך כל תו בנפרד, אפשר בעזרת שפות מסוימות להשתמש בפונקציות מובנות כמו `capitalize` או `title` שפועלות על כל המילה או המשפט. 

בגלים, המידע על אופן היישום מאחורי הקלעים פחות נגיש, מכיוון שהשפה מודולרית ומכוונת פונקציה. מימושים יכולים להשתנות בגרסאות שונות של השפה או ספריותיה.
