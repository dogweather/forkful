## מה ולמה? 

התחלת פרויקט חדש בתכנות מתרחשת כאשר משתמשים במשאבי קוד מתוך סביבת עבודה מסוימת (IDE) כדי ליצור אפליקציה או תוכנה חדשה. אנו מתחילים פרויקטים חדשים על מנת לממש רעיונות חדשים ולהוסיף תכנים או פיצ'רים שונים לתוכנות קיימות.

## איך לעשות:

הקוד הבא מדגים כיצד ליצור תוכנית "הללו עולם" בלוב של השפת C++:
```C++
#include<iostream> 

int main() 
{
    std::cout <<"הללו, עולם!"; 
    return 0; 
}
```
פלט (output):
```
הללו, עולם!
```
## בדילול עמוק:

### הקשר ההיסטורי
השפה C++, שנוצרה בשנת 1985 על ידי ביירן שטראוסטרופ, משמשת כבסיס לתכנות מונחה-עצמים ומהווה דיוקן בתחום. במהלך השנים, הבניית פרויקטים בקוד המקור המתקדם שלה הפכה לשטף לפיכך הקמת פרויקט חדש הפכה למתרגל הפשוט בעבודה היום-יום של המתכנתים.

### חלופות
חלופות ל C++ משנות ה-80 כוללות Python, Go, Rust, Swift, Java, ועוד. הבחירה בין השפות מתבצעת בהתאם לדרישות הפרויקט, המסגרת, ורמת הנוחות של התכנת.

### פרטי הישום
לאחר יצירת הפרויקט, חשוב לבחור במבנה הנכון של הקוד, לנהל את התלות, ולהשתמש בטכניקות מתאימות כמו ניהילת זכרון ומניפולציה של מחרוזות.
