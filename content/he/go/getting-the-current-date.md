## מה ולמה?
לקבל את התאריך הנוכחי בקוד זו דרך לדעת בדיוק מה היום במונחים של שנה, חודש, יום. מתכנתים עושים את זה לצורך לוגים, תזמונים ופונקציונליות שתלויה בזמן.

## איך לעשות:
נהלים בGo ישירים וקלים. ראה את הקוד למטה לדוגמא:

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	currentTime := time.Now()
	fmt.Println("Current Date: ", currentTime.Format("2006-01-02"))
}
```

פלט לדוגמא:

```
Current Date:  2023-03-15
```

## עיון מעמיק:
בGo, החבילה `time` משמשת לטיפול בזמנים ותאריכים. `time.Now()` מחזירה את הזמן הנוכחי. הפורמט `"2006-01-02"` הוא תבנית סטנדרטית בGo ומייצגת את הסדר שנה-חודש-יום.

עבר בעבר, פורמט התאריך והזמן היה תלוי במערכת הפעלה, שפת תכנות וספריות מסוימות. מאז הגדרת ה-C standard לתכנות תאריכים וזמנים, רוב השפות מאמצות גישה דומה.

בנוסף למתודה `Now`, יש ב-Go אפשרויות שונות לטיפול בתאריכים, כולל השוואות, חישוב הפרשים וניתוח מחרוזות לתאריך.

הטיפול בתאריכים צריך להיות מודע לאזור הזמן (Time Zone) בו המערכת פועלת, והחבילה `time` תומכת גם בזה.
